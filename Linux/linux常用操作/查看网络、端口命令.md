![ip信息](assets/1552463672993.png)

###### 2.查看端口

```shell
netstat -apn|grep 22
```

![查看端口](../开机关机/assets/1552458849471.png)

###### 3.列出所有处于监听状态的tcp端口`LISTEN`

```shell
netstat -lt
```

查看所有的端口信息, 包括 PID 和进程名称

```shell
netstat -tulpn
```

-a  all
-l  linsten
-n  显示ip地址
-p  程序名字programs
